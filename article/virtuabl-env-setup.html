Virtual Enveronmental Setup in python (11/10/2021 5:26 PM)

  1. You can use pip to install the virtual environment module in the system:

    pip install virtualenv

  2. Then test the installation by using the following:

    virtualenv --version

  3. Try creating a new virtual environment inside your project folder:

    `mkdir new-project-folder`
`cd new-project-folder`
`virtualenv new-project`

***This will create a folder in the current directory with a name new-

project .***

**If you want to create a virtual environment with a Python interpreter

of your choice as follows: ** virtualenv -p /usr/bin/python3 new-project

  1. You can activate this virtual environment with the following:

    source new-project/bin/activate

  2. If you have completed your work inside the virtual environment, you

    can deactivate and get out of the virtual environment with the following: deactivate
